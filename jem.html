Install jemdoc on Mac using Python 3:

  • Download Jemdoc Mathjax.

  • Unzip the file.

  • In Terminal (cd to your jemdoc folder first)

sudo cp jemdoc /usr/local/bin/jemdoc

chmod 755 /usr/local/bin/jemdoc
